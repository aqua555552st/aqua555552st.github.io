返回通知获取结果

作用

在切面类的返回通知方法上获取目标方法的执行返回结果。

这里查看：@AfterReturning源码：

@Retention(RetentionPolicy.RUNTIME)
@Target({ElementType.METHOD})
public @interface AfterReturning {
    String value() default "";

    String pointcut() default "";

    String returning() default "";//目标方法的执行返回结果

    String argNames() default "";
}

其中，returning表示：目标方法的执行返回结果。

2.应用实例

SmartAnimalAspect类[切面类] 的返回通知方法：showSuccessEndLog

//返回通知：把f2方法切入到目标方法正常执行完毕后的地方
/**
 * 解读：
 * 如果希望 目标方法(getSum)返回的结果 返回给 切入方法(showSuccessEndLog)
 * -> 在 @AfterReturning的属性加上:returning = "~"
 * -> 同时在 切入方法(showSuccessEndLog)的参数加上:Object ~
 * 本质上来说，returning = "res"就是接收invoke()函数的返回值
 */
@AfterReturning(value = "execution(public float SmartAnimalable.getSum(float, float))", returning = "res")
public void showSuccessEndLog(JoinPoint joinPoint, Object res) {
    Signature signature = joinPoint.getSignature();

    System.out.println("showSuccessEndLog(返回通知)-方法执行接收--日志--方法名:" 
            + signature.getName() + "-返回结果:" + res);
}
@Test
public void smartDogTestProxy() {
    //得到容器
    ApplicationContext ioc =
            new ClassPathXmlApplicationContext("beans.xml");

    SmartAnimalable sd = ioc.getBean("sd", SmartAnimalable.class);
    float sum = sd.getSum(101.0f, 11.2f);
}
切面类f1(前置通知)-日志--方法名:getSum--方法参数:[101.0, 11.2]
方法内部打印result = 112.2
showSuccessEndLog(返回通知)-方法执行接收--日志--方法名:getSum-返回结果:112.2//符合预期！
切面类f4(最终通知)-方法最终执行完毕--日志--方法名:getSum
