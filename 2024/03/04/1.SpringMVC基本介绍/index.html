1.SpringMVC基本介绍

特点&概述

快速入门

首先了解SpringMVC的(简易)登录流程：

需求说明：使用SpringMVC框架，完成一个最基本的测试案例，登录案例。

STEP1：创建项目，引入web框架，在WEB-INF包下创建lib目录，存放spring/springMVC所需jar包。

STEP2：创建src/applicationContext-mvc.xml文件(就是spring的容器文件)，文件名由程序员自定义。

STEP3：在web.xml配置文件中，配置中央/前端控制器DispatcherServlet

<!--web配置文件-->

<!--配置"中央控制器/前端控制器" -> 用户请求都会经过他的处理-->
<servlet>
    <servlet-name>springDispatcherServlet</servlet-name>
    <servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>
    <!--配置属性 -> contextConfigLocation：指定DispatcherServlet操作的spring配置文件-->
    <init-param>
        <param-name>contextConfigLocation</param-name>
        <param-value>classpath:applicationContext-mvc.xml</param-value>
    </init-param>
    
    <!--在web项目启动时，就自动加载DispatcherServlet-->
    <load-on-startup>1</load-on-startup>
</servlet>
<!--
    说明：
    1.这里的url-pattern是 / ，表示用户的请求都经过 中央控制器DispatcherServlet
    2.这样的配置也是rest风格的url请求
-->
<servlet-mapping>
    <servlet-name>springDispatcherServlet</servlet-name>
    <url-pattern>/</url-pattern>
</servlet-mapping>

STEP4：用jsp制作登陆页面login.jsp

<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<head>
    <title>登录</title>
</head>
<body>
<h3>登录页面</h3>
<%--回顾javaweb web工程路径专题
    action="login":表示url是 http://localhost:8080/springmvc/login
    action="/login":表示url是 http://localhost:8080/login
--%>
<form action="login">
    u:<input name="username" type="text"> <br/>
    p:<input name="password" type="password"><br/>
    <input type="submit" value="登录">
</form>
</body>
</html>

STEP5：定义UserServlet类

import org.springframework.web.bind.annotation.RequestMapping;

/**
 * 使用SpringMVC框架后，只需在类上标识 @Controller
 * 表示将该类视为一个控制器，注入到容器中
 */
@Controller
public class UserServlet {
    //编写方法响应用户请求
    //1.login()方法用于响应用户请求
    //2.@RequestMapping(value = "/login"):类似原生Servlet的url-pattern
    //3.当用户在浏览器输入 https//localhost:8080/web路过程径/login
    //  就能访问到login()
    //4.return：返回结果给视图解析器(InterResourceViewResolver)，视图解析器决定跳转到哪个页面
    @RequestMapping(value = "/login")
    public String login() {
        System.out.println("login ok...");
        return "login_ok";
    }
}

STEP6：用jsp制作登录成功login_ok.jsp页面（注意：文件名和@RequestMapping注解方法的返回值一样！）

<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<head>
    <title>登录成功</title>
</head>
<body>
<h1>恭喜 登录成功!</h1>
</body>
</html>

STEP8：配置applicationContext-mvc.xml文件

<!--spring容器配置文件-->

<!--1.配置注解自动扫描包-->
<context:component-scan base-package="com.lhy.web"/>

<!--2.配置视图解析器-->
<bean class="org.springframework.web.servlet.view.InternalResourceViewResolver">
    <!--配置 属性前缀prefix和后缀suffix-->
    <property name="prefix" value="/WEB-INF/pages/"/><!--从web目录开始-->
    <property name="suffix" value=".jsp"/>
    <!--根据UserServlet的login()方法，return "login_ok"；-->
    <!--就是转发到 /WEB-INF/pages/login_ok.jsp-->
</bean>

STEP9：配置tomcat，并指定web工程路径，运行

STEP10：浏览器输入http://localhost:8080/springmvc/login.jsp
