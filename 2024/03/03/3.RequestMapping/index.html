3.RequestMapping

基本使用

@RequestMapping注解可以指定控制器/处理器的某个方法的请求的url，基本用法在前文讲过了，RequestMapping：请求映射

修饰方法和类

@RequestMapping注解可以修饰方法，也可以修饰类。

当同时修饰类和方法时，请求的url就是/类请求值/方法请求值。

举例：

STEP1：在web/下创建请求页面request.jsp，在web/WEB-INF/pages/下创建返回页面seccess.jsp

<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<head>
    <title>购买商品</title>
</head>
<body>
<h1>购买商品</h1>
<%--解读：根据action解析的url:http://localhost:8080/工程路径/user/buy--%>
<form action="user/buy" method="post">
    购买人:<input type="text" name="username"><br>
    够买量:<input type="text" name="nums"><br>
    <input type="submit" value="购买">
</form>
</body>
</html>
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<head>
    <title>操作成功</title>
</head>
<body>
    <h1>恭喜，操作成功~</h1>
</body>
</html>

STEP2：编写 处理器/控制器UserHandler

@RequestMapping(value = "/user")
@Controller//Userhandler就是一个 处理器/控制器
public class UserHandler {
    /**
     * 解读
     * 1.method = RequestMethod.POST:表示请求buy目标方法必须是post
     * 2.RequestMethod 四个选项:POST GET PUT DELETE[后面详解]
     * 3.SpringMVC控制器 默认支持GET/POST两种方法
     *
     * 4.@RequestMapping同时标注在类和方法上
     * -> 请求的url就是：/类请求值/方法请求值，即：http://localhost:8080/工程路径/user/buy
     * @return
     */
    @RequestMapping(value = "/buy", method = RequestMethod.POST)
    public String buy() {
        System.out.println("购买商品");
        return "success";
    }
}

运行成功

指定请求方式

@RequestMapping注解可以指定请求的方式（GET/POST/PUT/DELETE）。html/jsp请求的方式(method)要和指定的一样，否则报错。

SpringMVC控制器(Controller/Handler)默认支持GET/POST，html/jsp不指定method，则只可以接收GET/POST。

若不匹配，则报405错误！

指定params和headers支持简单表达式

举例：

STEP1：在处理器UserHandler类中添加search方法，并在@RequestMapping注解中指定html/jsp的参数

@RequestMapping(value = "/user")
@Controller//Userhandler就是一个 处理器/控制器
public class UserHandler {
    /**
     * 解读
     * 1.method = RequestMethod.POST:表示请求buy目标方法必须是post
     * 2.RequestMethod 四个选项:POST GET PUT DELETE[后面详解]
     * 3.SpringMVC控制器 默认支持GET/POST两种方法
     *
     * 4.@RequestMapping同时标注在类和方法上
     * -> 请求的url就是：/类请求值/方法请求值，即：http://localhost:8080/工程路径/user/buy
     * @return
     */
    @RequestMapping(value = "/buy", method = RequestMethod.POST)
    public String buy() {
        System.out.println("购买商品");
        return "success";
    }

    /**
     * params = "bookId" 表示请求该目标方法时，必须给一个bookId参数，值没有限定
     * search(String bookId)：url携带的bookId值将作为search的参数
     * @param bookId
     * @return
     */
    @RequestMapping(value = "/find", method = RequestMethod.GET, params = "bookId")
    public String search(String bookId) {
        System.out.println("查询到的bookId为：" + bookId);
        return "success";
    }
}

支持Ant风格资源地址

举例

STEP1：在处理器UserHandler类中添加im方法

@RequestMapping(value = "/user")
@Controller//Userhandler就是一个 处理器/控制器
public class UserHandler {
	//无关代码...
	
    /**
     * 要求：可以配置 /user/message/aa, /user/message/aa/bb/cc
     * @return
     */
    @RequestMapping(value = "/message/**", method = RequestMethod.GET)
    public String im() {
        System.out.println("发送消息");
        return "success";
    }
}

STEP2：修改html/jsp的action，这里修改login.jsp

可配合$@PathVariable$映射URL绑定的占位符

举例

STEP1：在处理器UserHandler类中添加register方法

@RequestMapping(value = "/user")
@Controller//Userhandler就是一个 处理器/控制器
public class UserHandler {
	//无关代码...

    /**
     * 要求：处理器/控制器的目标方法能够获取username和userid
     * 页面的信息直接写在url中：http://ip:port/工程路径/user/reg/kristina/300
     * @return
     */
    @RequestMapping(value = "/reg/{username}/{userid}")//要和注解属性一一匹配！
    public String register(@PathVariable("username") String name,
                           @PathVariable("userid") String id) {
        System.out.println("接收到的参数--" + name + "username= " + id);
        return "success";
    }
}

STEP2：修改html/jsp的action，这里修改login.jsp

<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<head>
    <title>演示Ant风格的请求资源方式</title>
</head>
<body>
<h3>演示Ant风格的请求资源方式</h3>
<%--回顾javaweb web工程路径专题
    action="login":表示url是 http://localhost:8080/springmvc/login
    action="/login":表示url是 http://localhost:8080/login
--%>
<form action="user/message/aa/bb" method="get">
    u:<input name="username" type="text"> <br/>
    p:<input name="password" type="password"><br/>
    <input type="submit" value="登录">
</form>
<hr>

</body>
</html>

运行成功
